## Launch

- Clone this project

```bash
git clone https://github.com/KylinC/WhatTheHellBackingSchool.git
```

- install dependent packages (sever)

```
//install packages
cd WhatTheHellBackingSchool
pip install -r requirements.txt
```

- Run it on your server

```
python server.py
```
